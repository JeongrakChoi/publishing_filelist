# 웹 퍼블리셔 작업용 파일리스트  

## 개요  
- 주 사용자 : 웹 퍼블리셔, 웹 디자이너
- 설명  
  - 프로젝트시 협업 단계에서 프론트 화면을 보다 빠르게 공유하기 위함.  
  - 서버 환경없이 작업하는 소규모 프로젝트에 적합.

## 개발 환경
- Local (not server)
